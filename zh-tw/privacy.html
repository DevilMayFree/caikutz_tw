隱私權聲明

 

本政策聲明最新更新日期為2024年8月13日。

保護個人資訊與隱私是我們的最高要務。
我們深知您個人資訊的託付，我們將盡力不負所託，也將遵循以下原則，保障您的個人隱私。

依據個人資料保護法（以下稱個資法）第八條第一項規定，關於台端已提供或將提供本集團或本集團另行取得之個人資料（下稱「個人資料」），謹就本集團之蒐集、處理、利用、傳輸（包括國際傳輸）及揭露，向台端告知下列事項，請台端詳閱:

聯博證券投資信託股份有限公司及其關係企業，即個人資料控制者(統稱為“聯博”、“AB”、“我們”、“我們的” 及其他類似詞彙)，了解維護個人資料機密與安全的重要性。依個人隱私權相關法律規範，本聲明旨在說明聯博全球適用的個人資料隱私權做法，並提供透明的相關訊息。

以下說明我們所蒐集有關於您的個人資訊、我們與之共享個人資訊的單位、我們如何保護個人資訊、以及您如何就我們的個人隱私權聲明及做法、或是如何行使個人權利等方面與我們聯繫。聯博不會將您的個人資料販售予任何人。

我們的隱私權聲明反應當前處理個人資料的全球性準則及標準。這些準則包括了使用個人資料的通知、使用個人資料的選擇、資料取得、資料完整性、安全性、日後傳輸及執行/監管。我們遵循業務營運所在國家之隱私權相關法律。

目前、過去、潛在投資人也可能會在其投資文件的一部分中，看到個人隱私權的聲明，而在該等狀況下，該特定聲明即代表一般性的隱私權通知。本政策聲明一般來說同等公平適用於所有投資人，儘管某些章節係針對特定司法管轄區域投資人的揭露(例如歐盟、美國加州、加拿大或開曼群島)。若您是聯博目前員工、過去曾是員工、潛在員工或是外包商，適用之隱私權政策將會作為相關聘僱文件的一部分提供予您。

我們業已制定相關政策及程序，用以合理的保護個人資料的機密性及安全性；包括僅限獲適當授權人員才能取得非公開的個人資料，並針對個人資料採行普遍接受的實體、電子、程序性的保護措施。儘管本公司致力於個人資料的保護，我們無法完全保證您個人資料的安全性。

我們從不同來源蒐集各種公開及非公開的個人資料，包括了：

1.      我們透過開戶申請或其他方式所蒐集到的個人資料，包括了姓名、地址、電話、稅籍編號或其他政府辨別身分號碼(在適用法律允許或要求的情況下)、資產、收入、及其他家庭相關資料；

2.      有關與我們、本集團關聯公司及非相關之第三方(詳見下方所述)所進行交易之資料，其中可能包括了帳戶餘額及歷史交易明細；

3.      與聯博之間的溝通往來，包括但不限於提供所需之產品和服務時的往來電話及電子郵件；

4.      由公開或商用來源取得之資訊，例如個人姓名、地址、電子郵件、興趣和人口統計/概況等資料；

5.      由閉路電視鏡頭攝得之本公司辦公室訪客圖像；及

6.      造訪本公司網站者，透過線上表格、網站訪問數據和線上資料蒐集技術(例如 cookies、網路信標和像素)提供的相關資料，以及來自其他網站和服務所集結而成之公開可取得之資訊。我們的 cookie 政策可在此(僅備有英文版本)取得。我們目前不使用識別來自網頁瀏覽器“請勿跟蹤”信號的技術。

我們由多個來源所取得的資料，可能跟由我們直接向您取得或您造訪聯博網站的資料予以一併使用。

我們在必要或適當，並與商業用途相符時，使用所蒐集到關於您的個人資料，包括：   

+       履行合約義務

+       開立及管理帳戶

+       註冊用戶並依其請求得以進入網站或取得服務；

+       回應向我們提出的詢問或要求

+       滿足對產品或服務的要求

+       執行內部分析、運算、研發，用以優化及擴大我們的產品線，進一步針對不同區隔市場的推廣

+       發送有關我們網站、服務及產品的溝通和行政性質電子郵件

+       客製化並優化網站及我們服務的特色、效能和支援能力

+       針對網站及我方服務的使用者資料及互動數據進行分析、衡量、與研究

+       協助確認個人、帳戶、交易、訊息、或裝置是否與詐欺或其他惡意行為有關的可能性

+       積極管理我們的網站，係我們致力確保網站安全性的一環

+       提供客戶服務

+       作為聯博的金融犯罪及反貪腐計劃的一環，針對業務連絡窗口進行盡職調查

+       達成法律或法規監管義務及最佳實務做法的要求

+       調查並維護我們合法的權利

+       與企業商業交易行為有關之工作，像是資產或公司實體的出售、重組、或是清算

+       金融爭議處理（包括答辯、準備、參與及回應潛在之法律上請求、調查及政府機關之詢問）

+       滿足與前述任何一項相關的其他目的

 

我們蒐集台端個人資料有其法律依據，基於：(i)履行您為其中一方的合約，或應您請求提供服務、在簽訂合約前的必要動作；(ii)為了聯博的合法利益而進行的商業和直接行銷作為；或(iii)為了與您任職或其他方式表彰之實體開展業務行為，以增進聯博的合法利益，但前提是我們的合法利益不凌駕您的基本權利和自由。

在下文和本聲明中所述的情況下，我們可能會將前述所蒐集之任何個人資料提供給非關聯第三方。

若我們有義務或認為揭露或分享您的個人資料是適當的，我們可能會揭露您的個人資料，以遵循任何法律或法規要求；為調查、建立、執行、適用或履行相關條款和條件及其他合約，或為了保護我們的客戶、我們集團公司或他人的權利、財產或安全。聯博的金融犯罪及反貪腐計劃中，針對業務連繫窗口進行盡職調查，與其他公司及組織交換您的個人資料也包括在內。

我們使用數個第三方業者為我們的客戶提供服務(例如資料處理中心、用於開發和維護我們的網絡及系統的設施)。這些業者係扮演資料處理的角色，如何使用/不得使用您的個人資料，均受到合約嚴格的規範；不過我們仍負有保護您的個人資料之責。

若您想進一步了解履行合約義務的第三方業者等訊息，請聯繫聯博。

我們不會出售個人資料，或將該等個人數據傳輸給非關聯的第三方業者供其使用。前述方式蒐集之個人資料，在法律允許的情況下，基於行銷其產品或服務之目的，我們可能向關聯的投資、經紀、服務和保險公司揭露您的個人資料。選擇退出資料共享或是更動有關行銷活動的偏好，請利用下列方式，連絡帳戶代表或聯博服務人員(如適用)。

如欲從電子郵件發送清單中移除，只需取消訂閱。收到該等請求後，請容許本公司至少十(10)個營業日的作業時間，以完成移除之動作。

我們的網站可能連結到個人隱私權政策與我們不同的第三方業者網站，您應檢視這些網站的個人隱私權政策，以了解其蒐集、儲存、處理、傳輸及其他使用您個人資料的方式。

基於前述目的，本聲明所載之個人資料可能會傳輸至我們位在英國、歐洲經濟區和瑞士以外地區設立之關聯公司或非關聯的第三方，這些地區保護個人資料的程度不一。

我們將採取合理必要之措施，包括使用英國或歐盟相關主管機關(視狀況而定)所核准之契約範本，或採其他合約之做法，以確保個人資料的處理係安全嚴謹，相關做法係符合本聲明、歐盟一般資料保護法、英國資料保護法、及其他適用的個人隱私規定。

聯博由英國、歐洲經濟區和瑞士等地區傳輸個人資料之做法，係依循相關英國及歐盟資料保護主管當局所批准之契約範本條款所為。

我們僅和已有足夠個人資料保護措施的第三方業者，或者如適用時，獲得瑞士-歐盟隱私保護盾認證者、或同意適當保密協議條款或個人資料傳輸協議等，進行您的個人資料之分享，這將有助於確保先前提及的個人資料安全性。

您的權利: 

如果您居住在英國、歐洲經濟區或瑞士，則適用以下之權利。安道爾、阿根廷、澳洲、美國加州、加拿大、開曼群島、法羅群島、根西群島、香港、曼島、以色列、日本、澤西島、墨西哥、紐西蘭、新加坡、韓國、瑞士、烏拉圭和某些其他司法管轄區之個人，可能擁有某些個人資訊權利。您的權利將依當地法律及法規而有所不同。

請留意，依我們的資料保存政策，特別是為了遵循法律或法令之目的，或是選擇退出的要求，我們可能會保存您的個人資訊。

在我們提供任何資訊或進行任何更動之前，我們可能會要求您驗證個人身分及/或提供其他詳細資訊，用以協助我們回應您的請求。您也可以指定代理人、由其代表您提出請求，我們仍可能需要驗證您的身分、代理人任命及意圖行為的合法性。

您可能有權了解聯博持有關於您的資料、能取得該等資料，並能隨時更正該等資料。此外，您可能有權要求聯博刪除有關您的資料或是提供該等資料給您。

我們希望維持所持有關於您的資料之正確性，若您有更正之需要，請登入您的帳戶、或與帳戶窗口聯繫、或是和聯博服務人員接洽(如適用)。

您可能有權，基於個人情況的理由，隨時反對聯博處理您的個人資料。若您提出該等反對之請求，我們可能無法向您或您任職或其他表彰之實體提供所請求之服務。當您反對以直接行銷目的處理個人資料時，我們將不再以該等目的處理您的個人資料。

您可能有權以結構化、常用及機器可讀之格式，接收您提供給聯博的相關個人資料；基於您同意的前提、對特定類別資料明確表示同意、或依您與聯博所簽定之合約等條件下，您也有權將該等資料傳輸給另一個資料控制者，進行個人資料之處理，聯博將不予干涉。請留意個人資料可攜權僅適用於透過自動化方式之處理。

欲行使上述權利，可依本聲明中的連絡資訊，向適合的聯博辦公處所聯繫。我們將對您的請求保密，以合理方式努力提供聯博採行措施的相關訊息，並立即予以回應，在任何情況下，收到行使上述權利之請求後一個月內完成回覆。聯博將依相關法律要求，處理取得個人資料或更動個人資料之請求，我們不會針對任何行使隱私權的個人有差別待遇或進行報復。

向相關監理機構投訴之權利

若您認為個人資料的處理有違資料保護相關原則時，您有權向經常居住地區的成員國或是涉嫌侵權所在地的監理機構提出投訴。

向聯博提出投訴、問題或疑慮

有關本聲明或我們如何處理您的個人資料之任何問題、疑慮或投訴，均可以透過以下方式直接跟我們連絡。

收到投訴後，於合理的時間內，本公司代表將和您聯繫，向您說明關於個人資料之疑慮，並列舉如何解決疑慮的相關做法。我們將致力於確保您的投訴能即時並且妥善的解決。

聯博遵循廣泛的資料保存相關規定。該等要求係基於聯博所處司法管轄區和所經營受監管業務之相關法律和法規。除了詳載必須保存的記錄類型外，不同的法規亦明文規範該等資料必須保存的時間區段，某些狀況下，法規甚至規定業者需建立特定的方針，說明個人資料如何保存、以何種形式保存之做法。

聯博對個人資料保存期間之政策，依據營運、合約、法律、法規、稅務、稽核及/或歷史記錄等之要求，依取得個人資料之目的及運用，將以保存足夠之期間以符合前述相關之要求。

我們可能會不時更新我們的隱私權做法及本聲明。如果我們更改隱私權聲明，也將在網站公布修訂後之聲明並標明更新版本之日期。若我們對隱私權做法有重大之變動，以致於有重大影響隱私權聲明之情況時，我們將以郵件寄送複本、向您發送電子郵件或在本公司網站張貼橫幅等更為顯著的方式進行通知。
